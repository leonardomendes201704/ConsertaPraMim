# ConsertaPraMim - Admin Mobile Compacto

App mobile de operacao administrativa rapida, usando React + Vite + Capacitor.

## Scripts

- `npm run dev`: inicia servidor local.
- `npm run build`: gera build web.
- `npm run android:sync`: atualiza projeto Android com build web.
- `npm run android:apk:debug`: gera APK debug.

## Variaveis de ambiente

Crie `.env.android` com:

```
VITE_API_BASE_URL=http://SEU_IP_OU_DOMINIO:5193
```
